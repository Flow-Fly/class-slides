Higher Order Functions

Functions that take other functions as arguments or return them. Commonly used with array methods like forEach, map, filter, and reduce.

forEach

Executes a provided function once for each array element.


				myArray.forEach(function(element) {
				console.log(element);
				});

map

Creates a new array populated with the results of calling a provided function on every element in the calling array. 


				const doubled = myArray.map(num => num * 2);

filter

Creates a new array with all elements that pass the test implemented by the provided function. 


				const filtered = myArray.filter(num => num > 3);

reduce

Executes a reducer function on each element of the array, resulting in a single output value. 


				const total = myArray.reduce((acc, val) => acc + val, 0);

sort

Sorts the elements of an array in place and returns the sorted array. 


				myArray.sort((a, b) => a - b);

Advanced Sorting

Using localeCompare for sorting strings.


				wizards.sort((a, b) => a.lastName.localeCompare(b.lastName));
