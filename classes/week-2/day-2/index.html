Advanced Function Concepts

Deep dive into function expressions, arrow functions, passing functions, anonymous functions, and the arguments object.

Function Expression

A function expression assigns an anonymous function to a variable. 


			const mySayHiFunction = function(name) {
			return \`Hi \${name}\`;
			};

Arrow Function Syntax

Arrow functions provide a concise syntax for writing functions.


			const sum = (a, b) => a + b;

Passing Functions as Arguments

Functions can be passed as arguments to other functions.


			function morningRoutine(task) {
			task();
			}
			morningRoutine(() => console.log('Making coffee'));

Using Anonymous Functions

Anonymous functions are functions without a name, often used as arguments. 


			setTimeout(function() {
			console.log('Reminder: Take a break!');
			}, 1000);

The Arguments Object

Inside a function, 'arguments' is an array-like object containing all passed arguments. 


			function showArgs() {
			return Array.from(arguments).join(', ');
			}
			console.log(showArgs(1, 'blue', true));

Rest Parameters

Rest parameters allow a function to accept an indefinite number of arguments as an array. 


			const sumAll = (...nums) => {
				let sum = 0
				for (const elem of nums) {
					sum += elem
				}
				return sum
			}

Summary

Mastering these advanced function concepts enhances your ability to write flexible and powerful JavaScript code.
