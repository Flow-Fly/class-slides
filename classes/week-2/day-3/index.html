Introduction to the DOM

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

DOM Tree Representation

The DOM is represented as a tree of nodes, including elements, attributes, and text. These nodes can be manipulated with JavaScript.

Selecting Elements

Elements can be selected using document.getElementById(), querySelector(), and querySelectorAll().

Selecting Elements


        console.log(document.getElementById("main-title"))
        const title = document.getElementById("main-title")
        console.log(title.textContent)
