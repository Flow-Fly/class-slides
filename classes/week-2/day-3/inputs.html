Handling Input Fields

Create a grocery list item based on user input.


        button.addEventListener("click", () => {
          if (inputElement.value === "" ||
              quantityElement.value === "" ||
              quantityElement.value === "0") {
            alert("Wrong inputs")
            return
          }
          const li = document.createElement("li")
          li.textContent = \`\${inputElement.value}: \${quantityElement.value}\`
          list.append(li)
        })

Removing Items on Click

Click on a grocery list item to remove it.


        li.addEventListener("click", () => {
          li.remove()
        })

Listening to Input Changes

Update the title based on what is being typed in the input field. 


        inputElement.addEventListener("input", (event) => {
          title.textContent = event.target.value
        })
