Adding Event Listeners

Add an event listener to a button to create circles.


        buttonElement.addEventListener("click", (event) => {
          const circle = document.createElement("div")
          circle.classList.add("circle")
          circle.style.background = getRandomColor()
          sectionElement.append(circle)
        })

Event Object

Logging the event object to explore its properties.


        buttonElement.addEventListener("click", (event) => {
          console.log(event)
        })

Dynamic Element Interaction

Change circle color on mouse enter and remove it on click.


        circle.addEventListener("mouseenter", () => {
          circle.style.background = getRandomColor()
        })
        circle.addEventListener("click", () => {
          circle.remove()
        })

Resetting the DOM

Remove all circles from the section on button click.


        resetButton.addEventListener("click", removeCircles)
        function removeCircles(event) {
          sectionElement.innerHTML = ""
        }

Styling Multiple Elements

Apply random background color to all article elements.


        allArticles.forEach((article) => {
          article.style.background = getRandomColor()
        })

Creating Random Colors

Function to generate a random HSL color value.


        function getRandomColor() {
          return `hsl(${Math.random() * 360}, 80%, 60%)`
        }
