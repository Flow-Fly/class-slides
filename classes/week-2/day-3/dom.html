Introduction to the DOM

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

DOM Tree Representation

The DOM is represented as a tree of nodes, including elements, attributes, and text. These nodes can be manipulated with JavaScript.

Selecting Elements

Elements can be selected using document.getElementById(), querySelector(), and querySelectorAll().

Manipulating Content

Use innerHTML to change the HTML content and textContent to modify the text of an element.

Manipulating Properties and Style

Change properties directly to modify attributes and styles. For example, element.style.backgroundColor = 'yellow';

Creating and Appending Elements

Create new elements with document.createElement() and add them using parentNode.appendChild(childNode).

Modifying Elements

Elements can be modified or deleted using DOM methods. Use element.remove() to remove an element from the DOM.

Working with Attributes

Attributes can be read with getAttribute() and set with setAttribute(). For example, element.setAttribute('id', 'newId');

Event Handling

Assign events to elements using addEventListener. For instance, element.addEventListener('click', functionName);
