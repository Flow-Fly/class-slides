Flexbox Fundamentals

Flexbox offers a more efficient way to lay out, align, and distribute space among items in a container, even when their size is unknown or dynamic.

Important note

Flexbox have quite a few properties. Some target the parent / the container, others target the flex-items aka the childrens.

Main and Cross Axes

The main axis is the primary direction flex items are placed in the flex container, which can be horizontal (row) or vertical (column). The cross axis is perpendicular to the main axis and dictates how items align in the other direction.

Creating a Flex Container

To use Flexbox, first declare a flex container.


      .flex-container {
        display: flex;
      }

Direction of Flex Items

Control the layout direction of flex items with flex-direction.


      .flex-container {
        display: flex;
        flex-direction: row; /* or column, row-reverse, column-reverse */
      }

Justify Content

Align flex items along the main axis with justify-content.


      .flex-container {
        display: flex;
        justify-content: flex-start; /* flex-end, center, space-between, space-around */
      }

Align Items

Align flex items along the cross axis with align-items.


      .flex-container {
        display: flex;
        align-items: center; /* flex-start, flex-end, baseline, stretch */
      }

Flex Wrap

Allow flex items to wrap as needed with flex-wrap.


      .flex-container {
        display: flex;
        flex-wrap: wrap; /* nowrap, wrap-reverse */
      }

Flex Flow Shorthand

Use flex-flow to set flex-direction and flex-wrap together.


      .flex-container {
        display: flex;
        flex-flow: row wrap; /* column wrap, column nowrap, etc. */
      }

            .flex-container {
              display: flex;
              flex-wrap: wrap;
              align-content: space-between; /* flex-start, flex-end, center, space-around, stretch */
            }

Gap

This property oas it names implies allow you to add a gap between flex-items. 


            .flex-container {
              display: flex;
              gap: 1rem;
              justify-content: center;
            }

Ordering Flex Items

Change the visual order of flex items without affecting the DOM with the order property. 


      .item {
        order: number; /* default is 0 */
      }

Controlling Item Growth

Use flex-grow to define how much a flex item should grow relative to the rest of the flex items. 


      .item {
        flex-grow: 1; /* default is 0 */
      }

Controlling Item Shrinkage

flex-shrink determines how a flex item will shrink relative to others in the container. 


      .item {
        flex-shrink: 1; /* default is 1 */
      }

Aligning an Individual Item

Use align-self to override the container's align-items property for individual items. 


      .item {
        align-self: center; /* auto, flex-start, flex-end, baseline, stretch */
      }
