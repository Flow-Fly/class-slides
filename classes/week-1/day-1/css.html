Introduction to CSS

CSS (Cascading Style Sheets) styles the visual presentation of HTML documents.

In order to make your CSS file work, you need to link it to your html. 



<!DOCTYPE html>
<html lang="en">
	<head>
    <title>Linking HTML and CSS</title>    
</head>
<body>
  
</body>
</html>
            


<!DOCTYPE html>
<html lang="en">
	<head>
    <title>Linking HTML and CSS</title>
    <link rel="stylesheet" href="./my-css.css" /> 
</head>
<body>
  
</body>
</html>

CSS Selectors

Element Selector

Targets HTML elements directly.


      h1 {
        color: red;
      }

Class Selector

Targets elements with a specific class attribute. 


      .highlight {
        background-color: yellow;
      }

ID Selector

Targets an element with a specific id attribute. IDs are unique.

❗️ ID have a stronger specificity than class selector


      #main-title {
        font-size: 24px;
      }

Colors in CSS

Font & Background Colors

Colors can be set using names, HEX, RGB, or HSL values. 


      p {
        color: blue; /* Font color */
        background-color: #f0f0f0; /* Background color */
        background-color: rgb(127, 0, 54)
        background-color: hsl(124, 30%, 70%)
      }

CSS Units

px (Pixels)

Precise control over element sizing.

Best For: Exact sizing (borders, shadows).

% (Percentage)

Relative to the parent element's size, great for responsiveness.

Best For: elements that needs to have their size defined by the parent.

vh/vw (Viewport Height/Width)

Relative to the viewport's size for responsive designs.

Best For: elements that should take a specific amount of screen size.

rem (Root EM)

Relative to the root element's font-size, ideal for typography and spacing.

Best For: Font sizes, scalable spacing.

There's no one-size-fits-all answer. A combination of units tailored to your needs is often best:

  • Responsive Design: Use %, vh/vw, and rem for flexible layouts.
  • Accessibility: rem units help respect user settings for text size.
  • Precision: px units are useful for exact control where necessary.
