HTML Document Structure

An HTML document has a defined structure: <!DOCTYPE html>, <html>, <head>, and <body>. 

<!DOCTYPE html>
  <html>
  <head>
      <title>Page Title</title>
  </head>
  <body>
      <h1>My First Heading</h1>
      <p>My first paragraph.</p>
  </body>
  </html>

Inline vs Block Elements

Block Elements

Block elements start on a new line. E.g., <div>, <p>. 

<div>Block-level element</div>

Inline Elements

Inline elements don't start on a new line. E.g., <span>, <a>. 

<span>Inline element</span>

Creating a Simple Webpage

Combine block and inline elements to structure content.

<div>
      <h2>Page Title</h2>
      <p>This is a paragraph in a div block element. <a href="#">This is an inline link</a>.</p>
  </div>
