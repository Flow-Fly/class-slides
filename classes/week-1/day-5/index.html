Data Structures: Arrays and Objects

Understanding complex data handling by combining arrays and objects.

What is a Data Structure?

Data structures are ways to organize and store data in a computer so that it can be accessed and modified efficiently.

Combining Arrays and Objects

Arrays and objects can be nested within each other to store complex and structured data.

Example: Organizing Students by Age


				const students = [
					{ name: "Bob", age: 40 },
					{ name: "Alice", age: 40 },
					{ name: "John", age: 28 },
					{ name: "Toto", age: 26 }
				];
				
				const organizedStudents = {};
				for (const student of students) {
					if (!student.age in organizedStudents) {
						organizedStudents[student.age] = [student.name];
					} else {
						organizedStudents[student.age].push(student.name);
					}
				}

Here, students are grouped by age, demonstrating how to dynamically create and access keys and values in an object.

Complex Example: Organizing a Fridge


				const fridge = [
					{ name: "Tomatoes", category: "vegetable" },
					{ name: "Carrots", category: "vegetable" },
					{ name: "Duck", category: "meat" }
				];
				
				const organizedFridge = {};
				for (const food of fridge) {
					if (!food.category in organizedFridge) {
						organizedFridge[food.category] = [food.name];
					} else {
						organizedFridge[food.category].push(food.name);
					}
				}

This demonstrates organizing items in a fridge by category.
