Introduction to Jasmine for Testing

Learn how to write and run tests for your JavaScript code using Jasmine.

What is a Test Runner?

A test runner is a tool that executes your test suite and reports the results. It automates the process of running tests to ensure your code behaves as expected.

Writing Simple Tests

Before using a test runner, you can start by writing simple tests using console.log to assert the return values of functions. 


			function testSum() {
				console.log(sum(1, 2) === 3 ? "Pass" : "Fail");
			}
			testSum();

Setting Up Jasmine

To use Jasmine in the browser, include Jasmine's CSS and JavaScript files in your HTML. This sets up the environment for running Jasmine tests.

Writing Unit Tests with Jasmine

Unit tests check the smallest parts of an application in isolation (e.g., functions). Using Jasmine, we can write tests to automatically check our code's correctness.

Example: Testing the sum Function

Let's write tests for a sum function using Jasmine. 


			describe("Testing the sum function", function () {
				it("should add numbers together", function () {
						expect(sum(3, 5)).toBe(8);
				});
			});

Handling Errors in Tests

It's important to test not just for expected outcomes, but also how your code handles errors. 


			it("should throw an Error if an argument is not a number", function () {
				expect(function() { sum("chocolate", "tea"); }).toThrowError();
			});
