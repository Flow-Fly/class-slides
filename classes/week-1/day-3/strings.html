Introduction to Strings

Strings in JavaScript are used to represent and manipulate a sequence of characters.

  • Single quotes: ''
  • Double quotes: ""
  • Backticks (Template literals): ``
  • Constructor: new String()

String literals can include text within single quotes, double quotes, and backticks. Backticks allow for multi-line strings and embedding expressions.

String Concatenation and Template Literals

Combining strings:


      // Concatenation
      let myName = "Florian"
      const catName = "Illiu"
      const helloToYou = "Hello " + myName + "!"
      
      // Template literal
      const haikku = `In code we trust, vast,  
      Logic entwined, dreams compile,  
      Silent bugs whisper.`

      const byTheWay = `I have a cat, his name is ${catName}`

Working with String Methods

Strings come with a variety of methods for manipulation and querying.

  • Accessing characters: str[index], str.at(index)
  • Modifying case: str.toUpperCase(), str.toLowerCase()
  • Searching: str.includes(), str.replace('x', 'y'), replaceAll('a', 'b')
  • Splitting and joining: split(), join()

Immutability of Strings

Strings in JavaScript are immutable, meaning their values cannot be changed once they are created. 


      let myName = "Florian";
      myName[0] = "L"; // This will not change the string
      myName = "L" + myName.slice(1); // Correct way to "modify"

Comparing Strings

Strings can be compared using comparison operators or methods for case-insensitive checks. 


      // Case-sensitive comparison
      console.log("a" < "z");
      
      // Converting to common case for comparison
      const art = "art", butter = "Butter";
      console.log(art.toLowerCase() < butter.toLowerCase());

Type Coercion and String Operations

JavaScript performs type coercion in certain situations, like when using the == operator. 


      console.log(2 == "2"); // true, type coercion
      console.log(2 === "2"); // false, strict equality

Using === ensures that both the type and the value are the same, avoiding coercion.

String Splitting and Parsing

Strings can be split into arrays and then manipulated or analyzed. 


      const unparsedText = "Game 1: 3 blue, 4 red; ...";
      const separatedResult = unparsedText.split(":");
      console.log(separatedResult);

This is particularly useful for parsing structured string data.
