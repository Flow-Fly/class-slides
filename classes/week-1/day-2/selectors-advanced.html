Advanced CSS Selectors

Basic & Grouped Selectors

Select and style HTML elements. Group selectors to apply styles collectively. 


      h1, h2, .important {
        color: red;
      }

Combinator Selectors

Select elements based on their relationships.

      div > p {

Adjacent Sibling Selector (+)

Selects an element that is directly after another specific element. 


          /* Applies to any <p> element that directly follows an <h2> */

          h2 + p {
            color: blue;
          }

General Sibling Selector (~)

Selects all elements that are siblings of a specific element.


          /* Applies to all <p> elements that are siblings of <h2>, not just the next one */

          h2 ~ p {
            color: green;
          }

Example: All <p> elements that follow an <h2>, at any point, are selected and styled.

Attribute Selectors

Select elements based on attributes.


      input[type="text"] {
        border: 1px solid #000;
      }
      a[target="_blank"] {
        background-color: yellow;
      }

Pseudo-classes & Pseudo-elements

:hover

Style an element when it's hovered over.


      button:hover {
        background-color: green;
      }

:has

Select elements containing specific descendants.


      div:has(> p.special) {
        border: 2px solid blue;
      }

::selection

Style the portion of an element that is selected by a user.


      ::selection {
        color: white;
        background: black;
      }
