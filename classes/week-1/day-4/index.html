What is a Code Statement?

In JavaScript, a statement is an instruction to perform a specific action. Statements are the building blocks of a program, directing the computer to execute operations like calculations, data manipulation, and decision-making.

Loops and Conditions in JavaScript

Understanding the flow control with conditions and iterating with loops.

Conditional Statements

JavaScript offers several ways to control the flow of your code through conditional statements.

The if...else Statement

Execute code blocks based on a condition.


				if (condition) {
					// code to run if condition is true
				} else {
					// code to run if condition is false
				}

Logical Expressions in Conditional Statements

Logical expressions use operators to compare values, returning true or false. These expressions are crucial in making decisions in code. 


				if (age >= 18 && hasLicense) {
					console.log("Can drive");
				} else {
					console.log("Cannot drive");
				}

The switch Statement

Choose between multiple blocks of code to be executed.


				switch (expression) {
					case value1:
						// Statements executed when the result of expression matches value1
						break;
					case value2:
						// Statements executed when the result of expression matches value2
						break;
					default:
						// Statements executed when none of the values match the value of the expression
				}

for Loop

Iterate over a block of code a known number of times.


										for (initialization; condition; finalExpression) {
											// code to run on each iteration
										}

While and Do-While Loops

Loops that execute a block of code as long as a specified condition is true. 


			// While loop
			let i = 0;
			while (i < 5) {
				console.log(i);
				i++;
			}
			
			// Do-While loop
			let j = 0;
			do {
				console.log(j);
				j++;
			} while (j < 5);

Avoiding Infinite Loops

Ensure loop conditions will eventually become false. Be cautious with loop controls to prevent loops from running indefinitely. 


			// Correct loop condition
			for (let i = 0; i < 5; i++) {
				console.log(i);
			}
			
			// Potential infinite loop
			while (true) {
				// Make sure to have an exit condition
			}

Using Break and Continue

The break statement exits a loop early, and the continue statement skips the rest of the loop iteration and continues with the next one. 


			for (let i = 0; i < 10; i++) {
				if (i === 5) {
					break; // Exit loop when i is 5
				}
				if (i % 2 === 0) {
					continue; // Skip even numbers
				}
				console.log(i);
			}

for...of Loop

Iterate over iterable objects (including arrays, strings).


			for (const element of array) {
				console.log(element);
			}
