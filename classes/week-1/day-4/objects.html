Introduction to Objects

Objects are collections of key-value pairs. They are used to store various keyed collections and more complex entities.

Creating Objects


        const user = {
          name: "Bob"
        };

Use curly braces {} to create objects, with key-value pairs inside.

Accessing Object Properties


        const user = {
          name: "Bob"
        };
        console.log(user.name); // Dot notation
        console.log(user["name"]); // Bracket notation

Use dot notation or bracket notation to access properties of an object.

Adding Properties


        const user = {
          name: "Bob"
        };
        user.age = 30;
        // user : { name: "Bob", age: 30 }

Add new properties to an object by simply assigning a value to a new key.

Deleting Properties


        delete user.age;
        // user: { name: "Bob" }

Remove properties from an object using the delete keyword.

Listing Keys and Values


        console.log(Object.keys(user)); // Lists keys
        console.log(Object.values(user)); // Lists values

Use Object.keys() and Object.values() to list an object's keys and values.

Iterating Over Objects


        for (const key in user) {
          console.log(key, user[key]);
        }

Use a for...in loop to iterate over an object's properties.
