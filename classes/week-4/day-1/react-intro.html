JavaScript Frameworks and Libraries

  • Framework: provides a structure and set of guidelines to build your project
  • Library: provides reusable code to help solve common problems.

Why Use Them?

  • Speed up development process.
  • Provide well-tested solutions to common problems.
  • Improve code quality and maintainability.

What is React?

React is a declarative, efficient, and flexible JavaScript library for building user interfaces. It lets you compose complex UIs from small and isolated pieces of code called "components".

Main Features of React

  • Declarative: Design simple views for each state in your application, and React will efficiently update and render just the right components when your data changes.
  • Component-Based: Build encapsulated components that manage their own state, then compose them to make complex UIs.
  • Learn Once, Write Anywhere: Develop new features in React without rewriting existing code.

Components

Components let you split the UI into independent, reusable pieces, and think about each piece in isolation.

              }
              // Use the Title component in JSX
              function App() {
                return (
                  <div>
                    <Title colour="red" />
                  </div>
                )
              }
                

              // Define a Title component
              function Title(props) {
                return (
                  <>
                    <h2 style={{color: props.colour}} >{props.text}</h2>
                    <hr/>
                  </>
                )
              }
              // Use the Title component in JSX
              function App() {
                return (
                  <div>
                    <Title colour="red" text="Here is a title!" />

                    <Title colour="dodgerblue" text="And another one!" />
                  </div>
                )
              }
                

              // Define a Title component
              function Title(props) {
                return (
                  <>
                    <h2 style={{color: props.colour}}>{props.children}</h2>
                    <hr/>
                  </>
                )
              }

                )
              }

Creating a React Component

A React component can be defined as a JavaScript class or function. Components defined as classes have some additional features. 


        // Function component
        function Welcome(props) {
          return <h1>Hello, {props.name}</h1>;
        }
        
        // Class component
        class Welcome extends React.Component {
          render() {
            return <h1>Hello, {this.props.name}</h1>;
          }
        }
