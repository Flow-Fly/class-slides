About ChatGPT, copying code.

It's a thing to say that developers spend their time copy/pasting code from online ressources.

While true to some extent (you don't need to reinvent the wheel), it is a pretty bad thing to do.

Code is written in a language, therefore it is -to some extent- expressive.

Understanding by reading is an illusion, you will need to be able to create your own reasonning.

You need to do the jump!
