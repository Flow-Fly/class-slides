Useful commands

ls

              
              ls

Will list the content of the current directory

              
              ls ./documents

Will list the content of the documents folder if it exist

              
              ls ./documents/ironhack

Will list the content of the ironhack folder located in the documents folder

Important!

When writing a path / a filename, using TAB will autocomplete the name for you.

In the case of multiple results, you'll need to be more specific.

TAB is your friend.

Useful commands

cd (change directory)

              
              cd documents

Will take you to the documents folder (if it exist)

              
              cd

Will take you back to the home directory

              
              cd documents/ironhack
              cd doc[TAB]iro[TAB]

Will take you to the ironhack folder in the documents folder
