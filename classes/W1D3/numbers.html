Generating Random Integers: Step by Step

Let's break down the process of generating a random integer between 0 and 9, demonstrating the order of operations:

  1. Generate a random floating-point number between 0 (inclusive) and 1 (exclusive).
    • 0.26
  2. Multiply this number by 10 to scale it up.
    • 2.6
  3. Use Math.floor() to round down to the nearest whole number, ensuring it's an integer.
    • 2

                      const randomInt = Math.floor(Math.random() * 10)
                      console.log(randomInt)

This gives us a random integer from 0 to 9. Now, let's explore each step in detail:

Post vs. Pre Incrementation

  • Post-incrementation (variable++ or variable--)
  • Pre-incrementation (++variable or --variable)

        let postIncrement = 0;
        console.log(postIncrement++); // Prints 0, then increments
        console.log(postIncrement); // Prints 1
        
        let preIncrement = 0;
        console.log(++preIncrement); // Increments first, then prints 1
        console.log(preIncrement); // Prints 1

This distinction is a niche but it might prove useful sometime.
